Distant Unity

Distant Unity was a hackathon project for the internal IBM internhack 2020. It is a basic social network designed to help incoming first year university students meet and network with their peers and upperclassmen in ways they won't be able to as a result of the conditions imposed by the covid-19 pandemic. 

The app was written in pure HTML/CSS on the front end, with a flask backend and a PostgreSQL database supporting it. The app was deployed using IBMCloud;s cloud-foundry service, and the PostgreSQL database was likewise hosted on IBMCloud.

No github available (hosted on enterprise servers)

Date Your Diagnosis

Diagnostic Dating was a hackathon project for Deltahacks V. The project is a React Native app that allowed users to self-triage themselves be swiping left or right on symptoms to narrow down on what ailment the user is likely suffering from.

The front end is written as a React Native app that makes API calls to a Ruby-based backend, supported by a PostgreSQL database and deployed on heroku.

Mr. Pill

Mr. Pill was a hackathon project for Hack the North 2019. The project designed as a RESTFUL MERN app which would interface with a Google Home Mini through the Voiceflow API.
The user would tell their IOT device when they were taking their medication, and their service provider would get an update on the administrator webpage on how far through their prescription the patient has progressed.

Messenger Game Blocker

A chrome extension created to hide unwanted Facebook messages for the Facebook games specified by the user.

This extension uses Chrome synced storage to store user preferences, and hides Facebook messages from sources specified by the user by checking the page DOM on updates. The extensions also makes use of a background script to detect page refreshes and tab changes in order to set up action listeners on appropriate page elements on load.

Song To Pinyin

A python script written to convert the lyrics of Chinese songs from characters to the pinyin pronunciation system. The script pulls the lyrics from the requested song off of Genius.com through their API and runs the retrieved lyrics through the xpinyin python library to convert the retrieved characters and the desired pinyin.
